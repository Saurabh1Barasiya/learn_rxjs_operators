switchMap ki sabse simple definition (ek line mein):

Jo bhi naya value aaye, purane wale ka kaam cancel kar do aur sirf naye wale ka kaam karo!

Har baar naya input aaya → purana pending kaam cancel, sirf latest wala hi chalaao!

Real life example (bilkul simple):

  • Tu search box mein type kar raha hai:
  • "r" → API call shuru hua
  • "re" → "r" wala call cancel, "re" ka naya call shuru
  • "rea" → "re" wala cancel, "rea" ka call shuru
  • "react" → sirf iska result aayega, baaki sab cancel!
  • Toh switchMap = "Sirf latest hi boss hai, baaki sab ko bhool ja!"
