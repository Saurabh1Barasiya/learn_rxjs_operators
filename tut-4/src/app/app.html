lastValueFrom kya hota hai?

lastValueFrom() RxJS ka function hai jo:

  • ✔ Observable ko Promise me convert karta hai
  • ✔ Observable ke complete hone ka wait karta hai
  • ✔ Aur last emitted value return karta hai
  • Condition: Observable MUST complete. Agar complete nahi hua → Error aa jayega.
