"Jab bhi kisi bhi Observable me new value aaye, ye operator sabke latest values ko combine karke emit karta hai."

Initial Emit Rule: Output tabhi pehli baar emit karega jab har Observable ne kam se kam 1 value emit ki ho.

Latest Values Rule: Jab bhi koi Observable new value emit kare → output me sabhi Observables ki latest values combine ho kar aayengi.

Order Rule: Output me values ka order wahi rahega jo array me diya gaya hai.

Continuous Update Rule: Har naye value ke saath → output automatically update hota hai.
